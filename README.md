# Port Scanner

## Overview

A cross-platform command-line TCP port scanner written in C++ 20.

## Features

--all-ports (1-65535)
--speed <1|2|3|4|5>
--timeout <ms>
--banner
--output <txt|json>
--out-file <path>
--help
